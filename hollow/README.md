# Hollow

**Hollow** is a first-person psychological horror prototype made in **Unity 6**. The tagline: *the hollow adapts to you* — a creature that patrols, listens for sound, learns where you hide, and reroutes its hunt using NavMesh pathfinding and a small adaptive rules layer.

Unity does not keep the game files in this folder root. The actual project Unity Hub opens is:

**[`My project/`](./My%20project/)** — open **`My project`** in Unity Hub (full path: `ai-playground/hollow/My project`).

All setup, controls, systems, and optional FMOD / ML-Agents / Whisper notes live here:

**[→ Full documentation: `My project/README.md`](./My%20project/README.md)**

Quick pointer: after cloning, add the **`My project`** folder to Unity Hub, open it with **Unity 6000.x**, press **Play** (or start from **`MainMenu`** scene), then use **Play — Hollow** to enter the asylum blockout.
